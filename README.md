# RiverArt 3D Gaussian Splatting Viewer

A 3D Gaussian Splatting viewer for the RiverArt scene, designed to be embedded in X (Twitter) posts.

## Features

- Interactive 3D Gaussian Splatting viewer using PlayCanvas SuperSplat
- Optimized for embedding in X/Twitter feed
- High-quality rendering with antialiasing
- Auto-play functionality

## Files

- `index.html` - Main viewer page
- `player.html` - Player page for X embedding
- `index.js` - SuperSplat viewer JavaScript
- `index.css` - Viewer styles
- `settings.json` - Camera and rendering settings
- `assets/RiverArt_cutout.sog` - 3D model file

## Setup

1. Create a new repository on GitHub called `3D_Viewer_RiverArt`
2. Push this code to the repository
3. Enable GitHub Pages in the repository settings
4. The viewer will be available at `https://yourusername.github.io/3D_Viewer_RiverArt/`

## Usage in X

Post the URL: `https://yourusername.github.io/3D_Viewer_RiverArt/`

X will automatically detect the Twitter Card meta tags and embed the interactive 3D viewer.
